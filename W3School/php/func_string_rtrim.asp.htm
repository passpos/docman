PHP rtrim() 函数

PHP String 函数

实例

从字符串右侧移除字符：

<?php
$str = "Hello World!";
echo $str . "<br>";
echo rtrim($str,"World!");
?>

运行实例

定义和用法

rtrim() 函数移除字符串右侧的空白字符或其他预定义字符。

相关函数：

  • ltrim() - 移除字符串左侧的空白字符或其他预定义字符
  • trim() - 移除字符串两侧的空白字符或其他预定义字符

语法

rtrim(string,charlist)
参数 描述
string 必需。规定要检查的字符串。
charlist

可选。规定从字符串中删除哪些字符。如果省略，则移除下列所有字符：

  • "\0" - NULL
  • "\t" - 制表符
  • "\n" - 换行
  • "\x0B" - 垂直制表符
  • "\r" - 回车
  • " " - 空格

技术细节

返回值： 返回已修改的字符串。
PHP 版本： 4+
更新日志： 在 PHP 4.1 中，新增 charlist 参数。

更多实例

例子 1

移除字符串右侧的空格：

<?php
$str = "Hello World!    ";
echo "不使用 rtrim：" . $str;
echo "<br>";
echo "使用 rtrim：" . rtrim($str);
?>

以上代码的 HTML 输出如下（查看源代码）：

<!DOCTYPE html>
<html>
<body>

不使用 rtrim：Hello World!    <br>使用 rtrim：Hello World!
</body>
</html>

以上代码的浏览器输出：

不使用 rtrim：Hello World!
使用 rtrim：Hello World!

运行实例

例子 2

移除字符串右侧的换行符（\n）：

<?php
$str = "Hello World!\n\n\n";
echo "不使用 rtrim：" . $str;
echo "<br>";
echo "使用 rtrim：" . rtrim($str);
?>

以上代码的 HTML 输出如下（查看源代码）：

<!DOCTYPE html>
<html>
<body>

不使用 rtrim：Hello World!


<br>使用 rtrim：Hello World!
</body>
</html>

以上代码的浏览器输出：

不使用 rtrim：Hello World!
使用 rtrim：Hello World!
